# 特赞项目深度文案集

## atypica - 商业研究多智能体系统

### 产品概述

> **重新定义商业洞察的生产范式**

atypica 是首个面向商业研究的 Multi-Agent System，通过智能体分工协作，将传统的市场调研效率提升 300%。产品定位为"企业级商业洞察引擎"，服务于需要深度行业分析的战略决策场景。

---

### 产品洞察与机会

**发现痛点：**
- 传统商业研究报告周期长达 2-4 周
- 信息源碎片化，人工整合成本极高
- 初级研究员质量参差不齐，输出不可控
- 企业缺乏对竞争对手的实时监控能力

**市场机会：**
- 85% 的企业战略部门表示"需要更快速的市场洞察"
- AI Agent 技术成熟度达到可商用拐点
- 企业对"AI 分析师"接受度创历史新高

---

#### Chain-of-Thought 提示工程优化

**问题：** 早期版本 Agent 存在"幻觉输出"，缺乏可追溯性

**解决方案：**
```
提示词模板示例（分析推理 Agent）：

**任务：** 分析某品牌在社交媒体的声量变化

**思考步骤：**
1. 先列出收集到的数据源（必须真实可验证）
2. 说明数据的时间范围和采样方法
3. 计算声量指标（提及数、互动数、情感分）
4. 对比历史数据和竞品数据
5. 给出 3 个可能的解释假设
6. 标注不确定性等级（高/中/低）

**输出要求：**
- 每个结论必须引用数据源
- 标注置信度评分
- 提供原始数据链接
```

**成果：**
- Agent 准确率从 61% 提升至 **87%**
- 超行业平均水平 22 个百分点
- 客户投诉率下降 73%

## PitchLab - AI 驱动的表达能力训练平台

### 产品概述

> **让每一次表达都更有说服力**

发现 65% 的职场人因"表达焦虑"错失机会。PitchLab 通过 AI 打造"私人演讲教练"，提供实时反馈、智能改写与情景模拟训练， democratize 高端的演讲培训资源。

---

### 产品洞察与机会

**用户访谈发现（N=42）：**

| 痛点 | 占比 | 典型语录 |
|------|------|----------|
| 培训成本高 | 78% | "动辄几万的演讲课，性价比太低" |
| 缺乏即时反馈 | 82% | "录视频自己看，看不到问题在哪" |
| 无法量化进步 | 65% | "练了三个月，不知道有没有效果" |
| 临场应变差 | 71% | "写好的稿子还行，一被提问就慌" |

**市场机会：**
- 在线教育市场持续增长，演讲培训细分赛道尚无头部产品
- AI 语音识别与计算机视觉技术成熟，可低成本替代部分人工教练
- 远程办公常态化，线上表达需求激增

---

### 核心功能设计

#### 1. 实时反馈系统

**数据采集：**
- **语音分析**（Whisper API）
  - 语速检测（建议 180-200 字/分钟）
  - 停顿识别（过长/过短/位置不当）
  - 声音情感（紧张/自信/平淡）
  - 填充词统计（"嗯"、"啊"、"就是"）

- **视觉分析**（MediaPipe）
  - 眼神交流度（是否看镜头）
  - 肢体开放度（手势丰富度）
  - 面部表情（微笑/僵硬）
  - 站姿稳定性

**反馈呈现：**
```
┌─────────────────────────────────────┐
│  📊 本次表达评分：76/100              │
│                                      │
│  ✅ 语速合适（185 字/分钟）           │
│  ⚠️ 停顿过频（平均 8 秒/次，建议 15秒） │
│  ❌ 眼神不足（仅 34% 时间看镜头）     │
│  ✅ 手势丰富（12 次/分钟）            │
└─────────────────────────────────────┘
```

#### 2. 智能改写引擎

**金字塔原理优化：**
```
用户输入（粗稿）：
"我今天想讲一下我们的产品。它很棒，有很多功能。
首先是功能A，然后是功能B，还有功能C。
我们的团队也很厉害..."

AI 改写版本：

【版本 A：正式汇报风】
"我们的产品通过三大核心功能，解决客户痛点：
① 功能A：提升效率 50%
② 功能B：降低成本 30%
③ 功能C：增强体验
这背后是我们专业团队的持续打磨..."

【版本 B：生动故事风】
"想象一下，如果日常任务能节省一半时间...
我们的产品让这成为现实。它是如何做到的？..."

【版本 C：极简电梯风】（30秒版本）
"我们的产品通过三大功能，帮客户提效 50%降本 30%，
已被 100+ 企业采用。"
```

#### 3. 情景模拟训练

**50+ 场景库：**
- 🎯 投资路演
- 📊 汇报述职
- 🤝 客户提案
- 👔 求职面试
- 🎤 婚礼致辞

**AI 扮演机制：**
```
用户：（演讲中）

AI 听众：（突然提问）
"等等，你说的数据有来源吗？我对此表示怀疑。"

用户：（需临场回答）

"回答逻辑清晰，但可以增加'让我来解释一下数据来源'
这样的过渡句，显得更从容。"
```
